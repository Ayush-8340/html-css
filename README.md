# 🎬 YouTube Clone (HTML + CSS)

A simple **YouTube Clone** built using **HTML and CSS only**.  
This project replicates the basic layout and design of YouTube, including video thumbnails, titles, channel info, and a search bar — without any JavaScript functionality.

---

## 🚀 Features
- Responsive grid layout for video previews
- Video thumbnails with titles, author, and stats
- Channel profile picture aligned with video info
- Search bar styled like YouTube’s header
- Clean typography using Google Fonts (Roboto)

---

## 🛠️ Technologies Used
- **HTML5** → Structure of the webpage
- **CSS3** → Styling, layout, and responsiveness
